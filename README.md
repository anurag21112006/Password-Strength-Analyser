# Password Strength Analyzer

A Flask-based web application that evaluates password strength and provides security recommendations.

## Features

* Password strength analysis
* Password score calculation
* Security improvement suggestions
* Color-coded strength indicator
* Responsive web interface

## Technologies Used

* Python
* Flask
* HTML
* CSS

## How to Run

1. Install Flask:

   pip install flask

2. Run the application:

   python app.py

3. Open your browser and visit:

   http://127.0.0.1:5000
